# PhiFormat (WIP)

**A fanmade level format for the music game [*Phigros*](https://zh.moegirl.org.cn/Phigros).**

**Phigros 自制谱计划**

## 关于

PhiFormat 是整个 PhiX 项目的核心，旨在为 Phigros 自制谱计划提供一个统一的格式。

开始阅读规范文档：[GitHub Pages](https://phi-x.github.io/PhiFormat/)
